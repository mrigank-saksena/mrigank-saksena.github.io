Thinking With Data An Intuitive Approach to Data Science

A free platform to learn how to gain insights from data.

01

LEARNING
OBJECTIVES

  • DATA PREPROCESSING
    Learn how to import data from Excel using Python's pandas library, how to cleanse data, how to interpolate and deal with missing values, and how to turn raw data into something you can work with.
    DATA EXPLORATION
    Learn the skills to explore your data and determine key metrics. You'll also learn how to visualize your data and understand its characteristics.
    DATA MODELING
    Learn how to implement machine learing models using powerful Python libraries as well as how to gain insight from your models.
    DATA STORYTELLING
    Most importantly, learn how to take your analysis, understand it, and develop a comprehensive presentation using Visualization software like Tableau in order to make better data-driven decisions.
  • PYTHON
    Learn how to use Python for analyzing data.
    SQL
    Learn how to use SQL to query data.
    TABLEAU
    Learn how to create meaningful visualizations using Tableau.
    LIBRARIES
    Learn how to use libraries commonly used by data scientists including pandas, numpy, sklean, matplotlib, and more.
